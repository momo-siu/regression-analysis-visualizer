# 一元线性回归分析可视化工具

本项目是一个直观、互动的统计学教学演示工具，旨在通过可视化的方式帮助用户理解一元线性回归、相关系数以及变异分解等核心统计概念。

## 核心功能

- **实时交互式图表**：
  - **拖拽改点**：直接在图表中拖动数据点，实时观察回归线和统计指标的变化。
  - **自由增删**：支持在画布上点击添加新点，或删除现有异常点。
  - **等比例坐标系**：X 轴与 Y 轴单位长度严格一致，确保数据分布的视觉准确性。

- **深度统计分析**：
  - **动态相关性调整**：通过滑块设定目标相关系数 (R)，程序会自动变换数据点（保持均值和标准差不变），展示相关性的动态演变。
  - **Venn 图可视化**：通过重叠的圆面直观展示 $R^2$（决定系数），即 Y 的变异中有多少能被 X 所解释。
  - **变异分解图**：将总变异拆解为“解释的变异”与“未解释的变异”，帮助理解残差与回归的关系。

- **回归抽样模拟**：
  - **全过程仿真**：支持自定义总体参数（$\mu, \sigma, \beta$）进行大规模循环抽样（最高 10,000 次）。
  - **分布可视化**：自动绘制斜率 (b) 的抽样分布直方图，并叠加标准正态/t分布拟合曲线。
  - **重置机制**：一键恢复初始参数并清理仿真结果，便于多次实验对比。

## 核心逻辑实现说明

本项目不仅在视觉上追求精确，在底层逻辑上也严格遵循统计学原理：

### 1. 保持统计量不变的相关性变换
当用户拖动相关系数滑块时，系统并非简单地重新生成点，而是使用**正交分解**技术：
- 将原始 Y 向量分解为与 X 平行的分量和与 X 正交的分量（残差）。
- 按照目标 $R$ 重新合成 Y 向量。
- **逻辑效果**：在调整相关性的过程中，样本量 $n$、变量均值 $(\bar{x}, \bar{y})$ 和标准差 $(S_x, S_y)$ 保持绝对不变。

### 2. Venn 图面积与 $R^2$ 的精确映射
侧边栏 Venn 图中两个圆的重叠面积比例严格等于决定系数 $R^2$：
- 由于圆面重叠面积与圆心距离是非线性关系，程序采用**二分搜索法 (Binary Search)** 在数值上逼近精确的圆心坐标。
- **逻辑效果**：视觉上的“遮盖程度”真实反映了模型解释力。

### 3. 均衡分布的随机点生成
为了避免随机生成的点在视觉上出现过度“纵向”或“横向”的偏倚：
- **画布空间映射**：程序首先在标准化的“画布比例空间”生成分布，再根据当前图表的物理宽高比和坐标轴缩放量，逆向推算出数据空间中的分布参数。
- **逻辑效果**：无论窗口如何缩放，生成的点云在视觉上总是分布均衡，更符合直觉。

### 4. 视觉与数学的 1:1 对齐
- **坐标轴步长一致**：通过强制 ECharts 的 X 轴和 Y 轴物理像素单位长度相等。
- **逻辑效果**：回归线的斜率在视觉上的倾斜角度与数学上的数值完全对应（例如斜率为 1 时，线段与水平方向夹角正好为 45°）。

## 启动方式

本项目为纯前端实现，无需安装后端环境，但由于使用了 ES 模块，建议在本地服务器环境下运行：

1. **Python 用户**：在项目根目录运行 `python -m http.server 8000`，然后访问 `localhost:8000`。
2. **VS Code 用户**：安装 `Live Server` 插件，在 `index.html` 上点击“Open with Live Server”。

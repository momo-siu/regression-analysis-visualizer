# 一元线性回归分析可视化工具

本项目是一个直观、互动的统计学教学演示工具，旨在通过可视化的方式帮助用户理解一元线性回归、相关系数以及变异分解等核心统计概念。

## 核心功能

- **实时交互式图表**：
  - **拖拽改点**：直接在图表中拖动数据点，实时观察回归线和统计指标的变化。
  - **自由增删**：支持在画布上点击添加新点，或删除现有异常点。
  - **等比例坐标系**：X 轴与 Y 轴单位长度严格一致，确保数据分布的视觉准确性。

- **深度统计分析**：
  - **动态相关性调整**：通过滑块设定目标相关系数 (R)，程序会自动变换数据点（保持均值和标准差不变），展示相关性的动态演变。
  - **Venn 图可视化**：通过重叠的圆面直观展示 $R^2$（决定系数），即 Y 的变异中有多少能被 X 所解释。
  - **变异分解图**：将总变异拆解为“解释的变异”与“未解释的变异”，帮助理解残差与回归的关系。

- **完整数据展示**：
  - **统计指标面板**：实时计算均值、标准差、协方差、回归方程等指标。
  - **详细数据表**：展示每个点的坐标、预测值以及各项平方和计算细节。
  - **方差分析 (ANOVA)**：提供标准的变异分解表格。

## 启动方式

本项目为纯前端实现，无需安装后端环境，但由于使用了 ES 模块，建议在本地服务器环境下运行：

1. **Python 用户**：在项目根目录运行 `python -m http.server 8080`，然后访问 `localhost:8080`。
2. **VS Code 用户**：安装 `Live Server` 插件，在 `index.html` 上点击“Open with Live Server”。

## 目录结构

- `index.html`：主页面。
- `js/`：包含所有逻辑代码，按功能拆分为图表渲染、回归计算、交互处理等模块。
- `css/main.css`：定义了美观、现代的响应式界面布局。

## 技术方案

- **可视化**：Apache ECharts
- **公式渲染**：KaTeX
- **开发语言**：原生 JavaScript (ES6+), HTML5, CSS3

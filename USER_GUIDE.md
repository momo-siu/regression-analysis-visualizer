# 一元线性回归分析可视化工具 - 使用手册

本手册旨在帮助您快速上手并深入了解“一元线性回归分析可视化工具”的使用方法。

## 1. 运行说明

本项目采用原生 JavaScript (ES Module) 开发，无需复杂的环境配置。由于浏览器安全策略限制，ES 模块必须在 HTTP 服务器环境下运行，不能直接双击打开 HTML 文件。

### 环境要求
- 现代浏览器（推荐 Chrome, Edge 或 Firefox）
- 本地 HTTP 服务环境（以下任选其一）

### 启动命令

#### 方案 A：使用 Python (推荐)
如果您安装了 Python，在项目根目录下打开终端并运行：
```powershell
# Python 3.x
python -m http.server 8080
```
运行后，在浏览器访问：`http://localhost:8080`

#### 方案 B：使用 VS Code Live Server 插件
1. 在 VS Code 中打开项目文件夹。
2. 安装 `Live Server` 插件。
3. 右键点击 `index.html`，选择 **"Open with Live Server"**。

---

## 2. 核心交互指南

### 2.1 散点图操作 (主面板)
*   **拖拽改点 (默认模式)**：直接用鼠标选中并移动图中的蓝色散点，下方的回归线和统计指标会实时跟随变动。
*   **添加点**：在工具栏切换至“添加”模式，点击画布空白处即可增加新样本。
*   **删除点**：切换至“删除”模式，点击任意散点可将其移除。
*   **缩放视图**：使用工具栏的滑块调整坐标轴范围，工具会自动保持 X 和 Y 轴 1:1 的视觉比例。

### 2.2 动态相关性调整 (顶部栏)
*   拖动“目标相关系数 (R)”滑块，系统会应用**正交分解算法**变换现有数据点。
*   **注意**：此操作会改变点的相对位置以匹配目标 $R$，但会严格保持所有点的**均值**和**标准差**不变。

## 3. 统计分析说明

### 3.1 统计指标面板 (右侧)
*   展示均值、标准差、协方差以及回归方程。
*   **Venn 图**：左侧两个圆的重叠面积比例代表 $R^2$（决定系数），即 Y 的变异中有多少百分比可以由 X 解释。

### 3.2 变异分解图 (底部)
*   **总变异 (Y - Ȳ)**：蓝色虚线表示每个点到均值线的垂直距离。
*   **解释的变异 (Ŷ - Ȳ)**：蓝色虚线表示回归线上的预测值到均值线的距离。
*   **未解释的变异 (Y - Ŷ)**：蓝色虚线表示实际点到回归线的距离（残差）。
*   **视觉提示**：所有分解图中的散点已设为黑色小点，以便更清晰地观察蓝色辅助虚线。

## 4. 抽样模拟实验

### 4.1 参数配置
*   在底部“回归抽样模拟交互面板”中设置总体参数：
    *   $\mu_1, \sigma_1$：控制 X 的分布。
    *   $\mu_2, \sigma_2$：控制 Y 的分布。
    *   $\beta$：设置真实的回归系数。
    *   $n$：单次抽样的样本量。
    *   $R$：重复抽样次数（最高支持 10,000 次）。

### 4.2 结果分析
*   点击“分析”后，右侧会生成斜率 (b) 的分布图。
*   **直方图**展示了实验观测到的斜率分布。
*   **拟合曲线**展示了理论上的抽样分布。
*   **重置**：点击面板左上角的红色“↺ 重置”按钮，可一键恢复初始参数并清理所有分析图表。

---

## 5. 技术要点简述
*   **视觉一致性**：强制坐标轴物理比例 1:1，保证斜率 1 的直线看起来确实是 45 度。
*   **数据平滑**：随机生成逻辑考虑了画布比例，确保生成的点云在视觉上分布均衡，不会过度纵向或横向堆积。
